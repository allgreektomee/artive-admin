# JavaScript란 무엇인가?

IT 블로그를 시작하면서 가장 먼저 정리해볼 주제는 JavaScript다.  
JavaScript는 웹 페이지에 동적인 기능을 추가하기 위해 만들어진 프로그래밍 언어다.

HTML이 웹 페이지의 구조를 만들고, CSS가 디자인을 담당한다면, JavaScript는 사용자의 행동에 반응하고 화면을 바꾸는 역할을 한다.

## JavaScript가 필요한 이유

웹 페이지는 단순히 글과 이미지만 보여주는 정적인 문서에서 시작했다. 하지만 지금 우리가 사용하는 웹사이트는 훨씬 더 많은 일을 한다.

- 버튼을 클릭하면 메뉴가 열린다.
- 로그인 폼에 잘못된 값을 입력하면 경고 메시지가 나온다.
- 검색어를 입력하면 관련 결과가 바로 표시된다.
- 서버에서 데이터를 받아와 화면에 보여준다.

이런 동작을 가능하게 해주는 대표적인 언어가 JavaScript다.

## JavaScript는 어디에서 사용될까?

JavaScript는 처음에는 브라우저 안에서만 동작하는 언어였다. 하지만 지금은 사용 범위가 훨씬 넓어졌다.

- 프론트엔드 개발
- 백엔드 개발
- 모바일 앱 개발
- 데스크톱 앱 개발
- 자동화 스크립트

특히 Node.js가 등장하면서 JavaScript는 브라우저 밖에서도 실행할 수 있게 되었다. 그래서 하나의 언어로 화면 개발과 서버 개발을 모두 경험할 수 있다.

## JavaScript와 ECMAScript

JavaScript를 공부하다 보면 ES6, ES2015, ES2020 같은 표현을 자주 보게 된다.

ECMAScript는 JavaScript의 표준 규격이다. 쉽게 말하면 JavaScript가 어떤 문법과 기능을 가져야 하는지 정리해둔 약속이다.

ES6는 2015년에 발표된 큰 업데이트이며, 지금 JavaScript를 공부할 때 반드시 알아야 하는 중요한 문법들이 많이 포함되어 있다.

예를 들면 다음과 같은 기능들이 ES6에서 본격적으로 사용되기 시작했다.

```js
const name = "JavaScript";

const message = `Hello, ${name}`;

const numbers = [1, 2, 3];
const doubled = numbers.map((number) => number * 2);

console.log(message);
console.log(doubled);
```

실행 결과:

```text
Hello, JavaScript
[2, 4, 6]
```

위 코드에는 `const`, 템플릿 리터럴, 화살표 함수가 들어 있다. 앞으로 이 시리즈에서 하나씩 자세히 정리할 예정이다.

처음 보는 문법이 많아도 지금은 역할만 가볍게 보면 된다. `const`는 값을 담는 변수를 만들고, 백틱 문자열은 `${}` 안의 값을 문자열에 넣는다. `map()`은 배열의 각 값을 바꿔 새 배열을 만들고, `(number) => number * 2`는 짧게 쓴 함수다.

## JavaScript를 공부할 때 중요한 것

처음부터 모든 문법을 외우려고 하면 금방 지칠 수 있다. 중요한 것은 문법을 외우는 것이 아니라, 어떤 상황에서 왜 사용하는지 이해하는 것이다.

처음에는 다음 흐름으로 공부하면 좋다.

1. 변수와 자료형을 이해한다.
2. 조건문과 반복문으로 흐름을 제어한다.
3. 함수로 코드를 묶는 방법을 익힌다.
4. 스코프, 클로저, `this`처럼 JavaScript다운 개념을 이해한다.
5. ES6+ 문법으로 코드를 더 간결하게 작성한다.
6. 브라우저에서 실제로 동작하는 코드를 만들어본다.

## 정리

JavaScript는 웹 페이지에 동작을 추가하는 언어로 시작했지만, 지금은 프론트엔드와 백엔드 모두에서 사용되는 중요한 언어가 되었다.

앞으로 JavaScript 기본기부터 ES6+ 문법까지 하나씩 정리하면서, 단순히 문법을 외우는 것이 아니라 실제 개발에서 어떻게 쓰이는지 함께 살펴볼 예정이다.
